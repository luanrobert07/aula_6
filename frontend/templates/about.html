Sobre o Projeto

Informações sobre a proposta, a estrutura e o objetivo deste frontend desenvolvido em Flask.

Aplicação

Uma base clara para páginas web

Este projeto utiliza o Flask para criar uma interface web simples, com páginas separadas e navegação direta entre elas.

A aplicação possui rotas para Home, Sobre e Contato, servindo cada tela com render_template.

Componentes usados

  • Flask para criação das rotas
  • Jinja para renderização dos templates
  • Docker Compose para execução dos serviços
